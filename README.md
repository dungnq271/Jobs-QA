# RAG for multiple documents

## Supported Documents
- [x] .csv
- [ ] .pdf, .pptx
- [ ] .jpg, .png
  - [ ] ocr
  - [ ] description

## Setup
```
pip install -r requirements.txt
```

## Run
**Open 1 tab and run**:

```
./scripts/api.sh
```

**Wait for api startup complete and run in another tab**:
```
./scripts/app.sh
```
